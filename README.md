# Tournament Manager

Tournament Manager - это веб-приложение для управления турнирами по системе швейцарского турнира. Приложение позволяет создавать турниры, управлять участниками, генерировать пары для матчей и отслеживать результаты в реальном времени.

## Особенности

- Создание и настройка турниров
- Управление участниками их рейтингами
- Автоматическая генерация пар по системе швейцарского турнира
- Отслеживание результатов матчей
- Интерактивные таблицы standings
- Адаптивный интерфейс

## Технологии

- React
- TypeScript
- Vite
- Recharts (для визуализации данных)
- Lucide React (иконки)

## Запуск локально

**Зависимости:** Node.js

1. Установите зависимости:
   ```bash
   npm install
   ```

2. Запустите приложение:
   ```bash
   npm run dev
   ```

Приложение будет доступно по адресу [http://localhost:3000](http://localhost:3000)

## Сборка проекта

Для создания продакшн-сборки выполните:

```bash
npm run build
```

## Тестирование

Для запуска тестов:

```bash
npm run test
```

или в режиме watch:

```bash
npm run test:watch
```

## Публикация на GitHub Pages

Проект настроен для автоматической публикации на GitHub Pages при пуше в ветку `main` через GitHub Actions.
